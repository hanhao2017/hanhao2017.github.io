创建网站所在文件夹 hugo new site 文件夹名字

在hugo主页下载主题，下载到文件夹的themes下 git clone github仓库网址 themes/主题名 要输入密码

启动博客（此时可以打开博客网页了）（打开后什么都没有可能是配置文件的问题，看看主题的说明） hugo server -t 主题名 –buildDrafts

在github上建立空仓库

本地创建public文件 hugo –theme=toha –baseUrl="https://hanhao2017.github.io” –buildDrafts

进入public文件，将文件全部上传到空仓库中 git init git add . git commit -m “我的hugo博客第一次提交”

将本地public与github进行关联 git remote add origin https://github.com/hanhao2017/hanhao2017.github.io.git

git push -u origin master

​ ​ 解决 push 失败 rundll32.exe keymgr.dll,KRShowKeyMgr

两人同时fetch了一个分支。 第一个人修改后提交，第二个人提交就失败。 1.强制推送 $ git push -f

2.正常解决 先 git fetch origin 然后git merge origin/master, 和本地分支合并, 之后再push。
